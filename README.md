# pediwalk

Reseach dist word to word on wikipedia.
developed python 3

```sh
$ python pediwalk.py "ポケモン" "東京都"
search start ポケットモンスター -> 東京都
ポケットモンスター
1-UPスタジオ
---------result---------
ポケットモンスター
1-UPスタジオ
東京都
```

## Usage

```sh
$ python pediwalk.py [start] [goal]
```

## Requirement

```sh
$ pip install requests
```
